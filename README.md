# 🎓 Student Management System

A modern and responsive Student Management System developed using React.js. This application helps manage student records efficiently through a clean and user-friendly interface.

## 🚀 Features

- Add New Students
- View Student Records
- Update Student Information
- Delete Student Records
- Responsive Design
- Easy-to-Use Interface

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3

## 📂 Project Structure

```text
src/
├── components/
├── pages/
├── assets/
├── App.js
├── index.js
└── styles/
```

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/DhruvSharma189/Student-mangement-system-.git
```

### Navigate to Project Directory

```bash
cd Student-mangement-system-
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm start
```

The application will run on:

```text
http://localhost:3000
```
